# Educational Management Module

Module quản lý giáo dục cho Odoo - Quản lý khóa học, lớp học, giảng viên và học viên.

## Tính năng

- ✅ Quản lý khóa học và môn học
- ✅ Quản lý lớp học với workflow (Dự thảo → Mở → Kết thúc)
- ✅ Quản lý giảng viên và học viên
- ✅ Quản lý phòng học
- ✅ Tự động tính toán ngày kết thúc và % ghế đã đăng ký
- ✅ Báo cáo và phân tích (Pivot, Graph)
- ✅ Phân quyền User/Manager

## Cài đặt

1. Copy thư mục `edu_management` vào `addons/` hoặc `custom_addons/`
2. Restart Odoo server
3. Vào Apps → Update Apps List
4. Tìm "Educational Management" và Install

## Sử dụng

1. Tạo Môn học (Cấu hình → Môn học)
2. Tạo Khóa học (Khóa học)
3. Tạo Phòng học (Cấu hình → Phòng học)
4. Đánh dấu Partner là Giảng viên
5. Tạo Lớp học và quản lý workflow
